# next-codestore

This repository serves as a reference library of simplified implementations of commonly used features.

## 🛠️ Tech Stack

- **🌐 Framework:** Next.js
- **🧠 Language:** TypeScript
- **📦 Package Manager:** pnpm
- **🖥️ Runtime:** Node.js

## 🧩 Tooling

- **🎨 Styling:** Tailwind CSS
- **🔍 Spell Checking:** cspell
