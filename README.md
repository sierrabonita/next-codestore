# next-codestore

This repository serves as a reference library of simplified implementations of commonly used features.

## 🛠️ Tech Stack

- **🌐 Framework:** Next.js
- **🧠 Language:** TypeScript
- **📦 Package Manager:** pnpm
- **🖥️ Runtime:** Node.js
- **🎨 UI Library:** Chakra UI

## 🧩 Tooling

- **🔍 Spell Checking:** cspell
